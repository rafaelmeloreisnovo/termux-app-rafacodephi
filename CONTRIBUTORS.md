## How to Contribute

We welcome contributions from the community. To contribute to this fork:

1. **Read the License**: Understand the GPLv3 license and attribution requirements
2. **Fork the Repository**: Create your own fork
3. **Make Changes**: Follow the coding standards and conventions
4. **Submit Pull Request**: Include clear description of changes
5. **Sign-off Commits**: Use `git commit -s` to sign-off your commits

### Contribution Guidelines

- All contributions must comply with GPLv3 license
- Maintain existing code style and conventions
- Include appropriate copyright notices in new files
- Respect existing attributions and licenses
- Document significant changes
- Test your changes before submitting

### Copyright for Contributions

By contributing to this project, you agree that:
- Your contributions will be licensed under GPLv3
- You have the right to submit the contribution
- You understand that even minor contributions (including punctuation changes) are copyrightable works
