# Security Policy

## Fork Notice

This is a fork of the original Termux project. Security policies and procedures follow the upstream project.

- **Original Security Policy**: [termux.dev/security](https://termux.dev/security)
- **Upstream Repository**: [termux/termux-app](https://github.com/termux/termux-app)

---

## Reporting Vulnerabilities

Check https://termux.dev/security for info on Termux security policies and how to report vulnerabilities.

### For This Fork

If you discover security vulnerabilities specific to this fork:
1. **Do NOT** open a public issue
2. Contact the fork maintainer privately through GitHub security advisories
3. Provide detailed information about the vulnerability
4. Allow reasonable time for fixes before public disclosure

### For Upstream Issues

If the vulnerability affects the original Termux project:
- Report to the upstream Termux project following their security policy
- See [termux.dev/security](https://termux.dev/security) for details

---

## Security Acknowledgments

We acknowledge and follow the security practices established by the original Termux project and thank all security researchers who responsibly disclose vulnerabilities.
